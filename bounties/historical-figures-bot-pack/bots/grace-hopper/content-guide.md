## 5 Video Scripts

### 1: "We've Always Done It This Way"
**Visual:** Person stuck in old methods → innovation breaking through
**Script:** "The most dangerous phrase: 'We've always done it this way.' Change. Improve. Innovate. Or become obsolete. I did. You should too."

### 2: "The First Bug"
**Visual:** Moth in computer → tweezers removing it → modern "bug" term
**Script:** "1947: A moth stuck in Mark II. We removed it. I called it 'debugging.' Now every software issue is a 'bug.' Accidental legacy. True story."

### 3: "COBOL in English"
**Visual:** Machine code → COBOL code (English-like)
**Script:** "I created COBOL so humans could program in English. They said impossible. I did it. Then made it standard. Persistence wins. Always."

### 4: "A Ship in Port"
**Visual:** Ship docked → ship sailing into storm
**Script:** "A ship in port is safe. That's not what ships are built for. You in your comfort zone? Same thing. Sail. Risk. Grow."

### 5: "Ask Forgiveness, Not Permission"
**Visual:** Person waiting for approval → person acting → success
**Script:** "It's easier to ask forgiveness than permission. Innovation requires action. Bureaucracy kills ideas. Act first. Explain later."
